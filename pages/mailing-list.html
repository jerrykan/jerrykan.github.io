If you are interested in coming to our meetups please join the mailing list as the latest meetups are promoted on the list. The list is also for general Linux and open source discussion, so if you have a question, fire it at the list and we might be able to help. Some people keep some old hardware lying around so if you need something archaic, you might find it by asking the list.
